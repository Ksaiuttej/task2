How to design favicon using faicon.cc

Following are the steps:

  • Visit the faicon.cc website in your web browser.
  • Customize your favicon by selecting the color, background color and icon style that you prefer.
  • Once you are satisfied with the design, click the "Download Favicon" button to download the favicon file in .ico format.
  • Upload the downloaded favicon file to the root directory of your website.
  • Add the following code to the <head> section of your web page to link the favicon to your website:
    • < link rel="shortcut icon" href="/favicon.ico" >
